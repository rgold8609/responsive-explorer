So... You want to become an animator, huh?

That's a great idea, but not really an easy one. Because if you think art is hard then hoo boy is animation not even remotely in the same ball park. It's basically drawing but you need to understand physics. If that sounds like a nightmare beyond comprehension, don't worry because like physics, you can learn it just like anyone else. Unlike physics, you don't need to no math. Big win for non-masochists everywhere!
